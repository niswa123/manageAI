# МенеджAI

Автономный ИИ-управляющий смен для ресторанов. Снижает расходы на линейный ФОТ на 20% за 14 дней.

## Основные возможности
- **Интерактивный ROI Калькулятор**: расчет окупаемости и экономии в реальном времени.
- **Интеграция с Google Sheets**: сбор заявок и лидов напрямую в таблицу.
- **Полное соответствие 152-ФЗ**: кастомные согласия, политика конфиденциальности, cookie-баннер.
- **Интеграция с IIKO / r-keeper** (демо-логика).
